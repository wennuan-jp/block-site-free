# ZenBlock

A simple Chrome extension to block distracting websites.

## How to Install

1. Download this project.
2. Go to `chrome://extensions/` in your browser.
3. Turn on **Developer mode**.
4. Click **Load unpacked** and select this folder.

## How to Use

- **Block a site**: Click the extension icon while on a website and click "Confirm Block". The page will refresh and show a blocked message.
- **Manage list**: Right-click the icon and go to **Options** to add or remove websites from your block list.
